# Documentación de Componentes Comunes

## Estructura
Los componentes comunes se encuentran en `src/components/common/` y proporcionan funcionalidad reutilizable en toda la aplicación.

### CustomSurface.tsx
- **Propósito**: Proporciona un contenedor visual consistente para elementos de la interfaz
- **Funcionalidad**:
  - Estilización consistente de superficies
  - Manejo de sombras y elevación
  - Adaptación a diferentes temas

### Login.jsx
- **Propósito**: Gestiona la autenticación de usuarios
- **Funcionalidad**:
  - Formulario de inicio de sesión
  - Integración con Supabase
  - Manejo de estados de autenticación

## Integración con el Sistema
- Utilización en múltiples vistas
- Consistencia visual en la aplicación
- Manejo de estados globales

## Consideraciones Técnicas
- Reutilización de componentes
- Optimización de rendimiento
- Manejo de errores de autenticación
